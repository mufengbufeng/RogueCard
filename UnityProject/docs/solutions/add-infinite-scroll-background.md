# 解决方案：无限滚动背景（2D SpriteRenderer）

## 问题
飞机大战固定相机玩法缺少持续向下滚动的背景，视觉表现不连贯，且需要可调速度以便配合节奏与表现。

## 根因
场景中缺少可复用的“背景滚动组件”和稳定的拼接/重置逻辑，导致无法实现平滑循环。

## 解决方案
- 新增背景滚动组件：基于 `SpriteRenderer.bounds` 计算单张高度，按纵向向下循环移动。
- 通过两张或多张背景拼接，达到无缝循环效果。
- 支持运行时调整速度（含 0 速度暂停）与动态切换背景 Sprite，并在切换后重建布局。

## 结果
PlayMode 验证通过：背景无缝循环，速度调整与切换 Sprite 后滚动保持连续。
